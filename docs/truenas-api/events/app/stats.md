# app.stats

## Summary

Retrieve statistics of apps.

This event is generated by a dynamic source. If you want to specify subscription params, subscribe to it using the name in the following format `app.stats:{"param": "value"}`

## Required Roles

- `APPS_READ`

## Schema

- Type: object

### Subscription parameters

- Schema name: `AppStatsEventSourceArgs`
- Type: object
- No Additional Properties
#### interval

- Schema name: `Interval`
- Type: integer
- Default: 2

Interval in seconds between statistics updates.
- Value must be greater or equal to `2`

### ADDED

- Schema name: `AppStatsEventSourceEvent`
- Type: object
- No Additional Properties
#### fields (required)

- Schema name: `Fields`
- Type: array of object
- No Additional Items

##### Each item of this array must be:

##### AppStatsEventSourceEventItem

- Schema name: `AppStatsEventSourceEventItem`
- Type: object
- No Additional Properties
###### app_name (required)

- Schema name: `App Name`
- Type: string

Name of the application these statistics are for.

###### cpu_usage (required)

- Schema name: `Cpu Usage`
- Type: integer

Percentage of cpu used by an app.

###### memory (required)

- Schema name: `Memory`
- Type: integer

Current memory (in bytes) used by an app.

###### networks (required)

- Schema name: `Networks`
- Type: array of object

Array of network interface statistics for the application.
- No Additional Items

####### Each item of this array must be:

####### AppStatsEventSourceEventItemNetwork

- Schema name: `AppStatsEventSourceEventItemNetwork`
- Type: object
- No Additional Properties
######## interface_name (required)

- Schema name: `Interface Name`
- Type: string

Name of the interface used by the app.

######## rx_bytes (required)

- Schema name: `Rx Bytes`
- Type: integer

Received bytes/s by an interface.

######## tx_bytes (required)

- Schema name: `Tx Bytes`
- Type: integer

Transmitted bytes/s by an interface.

###### blkio (required)

- Schema name: `AppStatsEventSourceEventItemBlkio`
- Type: object

Block I/O statistics for the application.
- No Additional Properties
####### read (required)

- Schema name: `Read`
- Type: integer

Blkio read bytes.

####### write (required)

- Schema name: `Write`
- Type: integer

Blkio write bytes.
